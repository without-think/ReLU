# 📍 Smart Sejong 프론트엔드 페이지 경로 목록

## 🔐 인증 페이지 (비로그인 접근 가능)

### 로그인 페이지
```
http://localhost:3000/login
```
- 카카오 로그인
- 세종대 포털 인증

---

## 🏠 메인 페이지 (로그인 필요)

### 루트 페이지
```
http://localhost:3000/
```
- 자동으로 `/learning`으로 리다이렉트됩니다

---

## 📚 학습 현황 페이지
```
http://localhost:3000/learning
```
- 성적표 CSV/엑셀 업로드
- 이수 학점 대시보드 (전공/교양/총 학점)
- 기이수 과목 리스트

---

## ✨ AI 시간표 추천 페이지
```
http://localhost:3000/recommendation
```
- 필터 설정 (공강 요일, 최소 공강 횟수, 시간대 선호, 필수 포함 과목)
- AI 추천 조합 생성
- 핀 고정 기능

---

## 📅 내 시간표 페이지
```
http://localhost:3000/timetable
```
- 시간표 스냅샷 저장/불러오기
- 보관함 관리
- 시간표 편집 및 삭제

---

## 👥 그룹 협동 페이지
```
http://localhost:3000/group
```
- 그룹 생성 및 초대 코드 공유
- 그룹 참가
- 멤버 시간표 실시간 조회
- 추천 과목 복사

---

## 👤 프로필 관리 페이지
```
http://localhost:3000/profile
```
- 개인 정보 조회/수정
- 인증 상태 확인

---

## 📋 전체 경로 요약

| 경로 | 설명 | 인증 필요 |
|------|------|----------|
| `/login` | 로그인 및 포털 인증 | ❌ |
| `/` | 루트 (자동 리다이렉트) | ✅ |
| `/learning` | 학습 현황 | ✅ |
| `/recommendation` | AI 시간표 추천 | ✅ |
| `/timetable` | 내 시간표 | ✅ |
| `/group` | 그룹 협동 | ✅ |
| `/profile` | 프로필 관리 | ✅ |

---

## 🔒 인증 처리

- **비로그인 상태**: `/login`으로 자동 리다이렉트
- **로그인 상태**: 모든 페이지 접근 가능
- **토큰 저장**: localStorage에 JWT 토큰 저장

---

## 💡 참고사항

- 모든 경로는 `http://localhost:3000`을 기본으로 합니다
- 개발 서버 실행 시 포트가 다르면 해당 포트로 접속하세요
- 백엔드 API는 `http://localhost:8080`에서 실행되어야 합니다
