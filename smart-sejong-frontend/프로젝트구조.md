# 프로젝트 구조 및 실행 가이드

## 📁 프로젝트 구조

```
창의학기제/
├── smart-sejong-backend/  # 백엔드 서버 (Spring Boot)
│   ├── src/
│   ├── build.gradle
│   ├── settings.gradle
│   └── ...
└── smart-sejong-frontend/         # 프론트엔드 (React + TypeScript)
    ├── src/
    ├── package.json
    └── ...
```

## 🚀 실행 방법

### 1. 백엔드 실행

```bash
# 백엔드 디렉토리로 이동
cd smart-sejong-backend

# Gradle로 실행
./gradlew bootRun
# 또는 Windows
gradlew.bat bootRun
```

백엔드는 `http://localhost:8080`에서 실행됩니다.

### 2. 프론트엔드 실행

```bash
# 프론트엔드 디렉토리로 이동
cd smart-sejong-frontend

# 의존성 설치 (최초 1회만)
npm install

# 개발 서버 실행
npm run dev
```

프론트엔드는 `http://localhost:3000`에서 실행됩니다.

## 🔗 연동 설정

- **프론트엔드**: `http://localhost:3000`
- **백엔드**: `http://localhost:8080`
- **프록시**: Vite가 자동으로 `/api/*` 요청을 백엔드로 프록시합니다.

## ⚙️ 환경 변수

프론트엔드에서 백엔드 주소를 변경하려면 `.env` 파일 생성:

```env
VITE_API_BASE_URL=http://localhost:8080
```

## 📝 참고사항

- 백엔드와 프론트엔드는 독립적으로 실행됩니다.
- 백엔드를 먼저 실행한 후 프론트엔드를 실행하는 것을 권장합니다.
- CORS 설정은 백엔드의 `SecurityConfig.java`에서 관리됩니다.
