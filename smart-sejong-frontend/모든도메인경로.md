## 🔄 자동 리다이렉트 경로

### 잘못된 경로 처리
다음과 같은 잘못된 경로들은 자동으로 루트(`/`)로 리다이렉트됩니다:

- `/login/learning` → `/` → `/learning`
- `/login/anything` → `/`
- `/존재하지않는경로` → `/`
- 기타 모든 잘못된 경로 → `/`

---

## 🧪 테스트 방법

### 1. 직접 URL 입력 테스트
브라우저 주소창에 직접 입력:
```
http://localhost:3000/learning
http://localhost:3000/recommendation
http://localhost:3000/timetable
http://localhost:3000/group
http://localhost:3000/profile
```

### 2. 네비게이션 메뉴 클릭
- 상단 헤더의 네비게이션 메뉴 클릭
- 모바일 하단 네비게이션 바 클릭

### 3. 잘못된 경로 테스트
```
http://localhost:3000/login/learning
http://localhost:3000/잘못된경로
```
→ 자동으로 루트로 리다이렉트됨

## ⚠️ 주의사항

1. **백엔드 연동**: 모든 페이지는 백엔드 API와 연동됩니다
   - 백엔드가 실행되지 않으면 데이터는 표시되지 않지만 UI는 정상 작동합니다

2. **인증 상태**: 
   - 개발 모드: 모든 페이지 접근 가능 (인증 우회)
   - 프로덕션: 로그인 필요

3. **포트**: 기본 포트는 `3000`입니다
   - 다른 포트 사용 시 `vite.config.ts`에서 변경 가능
