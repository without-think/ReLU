# 개발 모드 설정 가이드

## 🔓 인증 우회 설정

개발 중에는 로그인 없이 모든 페이지에 접근할 수 있도록 설정되어 있습니다.

### 현재 설정
- **개발 모드 (`npm run dev`)**: 인증 없이 모든 페이지 접근 가능 ✅
- **프로덕션 빌드**: 인증 필요

### 인증 우회 비활성화 방법

인증을 다시 활성화하려면 환경 변수를 설정하세요:

**방법 1: .env 파일 생성**
```env
VITE_BYPASS_AUTH=false
```

**방법 2: package.json 스크립트 수정**
```json
{
  "scripts": {
    "dev": "VITE_BYPASS_AUTH=false vite"
  }
}
```

## 📝 참고사항

- 백엔드의 `SecurityConfig.java`는 이미 `.anyRequest().permitAll()`로 설정되어 있어 모든 API 요청이 허용됩니다.
- 프론트엔드에서만 인증 체크를 하고 있었는데, 개발 모드에서는 이를 우회하도록 수정했습니다.
- 프로덕션 배포 시에는 반드시 인증을 활성화하세요!
